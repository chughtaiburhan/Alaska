# Alaska - Kenai River Trophy Fishing Website

A responsive Next.js 13 project showcasing **trophy fishing** experiences on the Kenai River. Includes animated sections, a mobile-friendly carousel, dynamic imports, and a clean UI built with **Tailwind CSS**, **Framer Motion**, and **Next.js Image optimization**.
